### Aapke Project ke liye **Best Scope** (Resume ke liye perfect):

**Abhi focus karo in 8 cheezon pe:**

1. Create New Account
2. Get Account Details
3. Check Balance
4. Deposit
5. Withdraw
6. Transfer Money (with proper debit + credit entry)
7. Transaction History (by account)
8. Update Account (name, type etc.) — optional

Yeh 8 cheezein agar achhe se kaam kar rahi hongi + clean code + proper transaction table, toh aapka project **medium level** ke liye bahut strong dikhega.
