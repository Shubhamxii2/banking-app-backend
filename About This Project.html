Shu Bank

Secure & Modern Banking API using Spring Boot

Current Features Implemented

  • ✅ Create New Bank Account
  • ✅ Get Single Account Details
  • ✅ Get All Accounts
  • ✅ Check Account Balance
  • ✅ Deposit Money
  • ✅ Withdraw Money
  • ✅ Fund Transfer (User to User)
  • ✅ Transaction History by Account
  • ✅ Get Single Transaction by ID
  • ✅ Mini Statement (Last 5 transactions)
  • ✅ Full Statement (with totals)
  • ✅ Full Statement as PDF Download
  • ✅ Update Account Details
  • ✅ Delete Account

Recently Added

Fixed Deposit (FD) Creation – Users can now open Fixed Deposits with proper interest calculation and maturity tracking.

Tech Stack

  • Spring Boot 3+
  • Spring Data JPA + Hibernate
  • MySQL Database
  • Lombok
  • iText 8 (for PDF generation)
  • Tomcat Embedded Server
